Bathymetry Viewer based on Cloud Optimized Tiff (COG) and Vector Tiles

Derived product from the GEBCO 2021 Grid, made with NaturalEarth.

Check out Download Bathymetry site for the background information on processing.

To get rid of backend components (database, mapservers, web application containers) modern geodata formats like vector tiles and cloud optimized tiffs (cog) were tested.

Check out github.com/OpenDEM/BathymetryViewer_COG for more information about the results.
